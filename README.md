# DigitalHunter2

A system for performing queries and graphical presentation based on data

## Run the app

From the project root directory, in a **separate terminal** (with the venv activated):

```bash
docker compose up --build -d  
```

## Usage

In the browser, open the link 🔗 [app](http://localhost:8000/docs)

A list of all functions ready for testing will appear.
